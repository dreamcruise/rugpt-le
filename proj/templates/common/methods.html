Транскрибация аудио в ChatGPT

Примеры запросов:

Python

                import requests

                file = open('ttr1.mp3', 'rb')
                req = requests.post(
                        url='https://www.fmt-ai.com/stt',
                        headers={
                            'Authorization': "ZTlt9Kqd2sFHqdwKiaR4TaDxU2vVCIFMhmCnb7Qk5WRzWJEc14y"
                        },

                        data={
                            'model': "whisper-1"
                        },
                        files={
                            'file': file
                        }
                )

Ответ

                Сервер отправляет ответ,  в втрибуте text которого содержится транскрибация
                'Да, на Титане, крупнейшем спутнике Сатурна, действительно существуют осадки, которые можно сравнить с дождём. Однако это не водяной дождь, как мы его знаем на Земле. На Титане идёт дождь из метана и этана, так как эти вещества находятся в жидком состоянии при температурах, характерных для Титана, около 180 градусов Цельсия. Метановые дожди могут образовываться в результате конденсации метана в атмосфере, и некоторые исследования показали, что такие осадки могут происходить в результате сезонных изменений. На Титане также имеются реки и озёра из метана и этана, что создаёт уникальный климатический и геологический цикл, отличающийся от земного.'
